# YouTube Subtitles Loader

Java CLI приложение для загрузки и очистки субтитров из YouTube видео.

## Описание

Приложение использует встроенную утилиту `yt-dlp` для скачивания субтитров с YouTube.
Автоматически выбирает лучшие доступные субтитры по приоритету:
1. Русские ручные
2. Русские автоматические
3. Английские ручные
4. Английские автоматические

Очищает субтитры от VTT-тегов, временных меток и дубликатов, возвращая чистый текст.

## Запуск

### 1. Сборка проекта

```bash
./gradlew clean build
```

### 2. Запуск CLI

```bash
java -jar build/libs/yt-subtitles-loader-*.jar "https://www.youtube.com/watch?v=VIDEO_ID"
```
